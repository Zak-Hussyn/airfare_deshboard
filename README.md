Bangladesh Flight Fare Analysis And Predictor Dashboard
Following my previous project “Airline Insights & Fare Prediction”,  I have now taken it a step further.  I developed an interactive dashboard using Streamlit to transform the prediction model into a complete analytical tool. This dashboard predicts fares and allows users to explore underlying patterns in airfare data.

Dashboard Highlights:
- Real-Time Fare Prediction
- Key Metrics
- Section A: Travel Class Analysis
- Section B: Stopover Impact on Airfare
- Section C: Seasonality Insights

For fare prediction, enter the details. Tax & Surcharge will be calculated automatically (15%) once you enter the Base Fare. For Base Fare, if you need assistance, use Google Flights. The link can be found just below the "predict" button.
Three filters are given on the left side bar. Use Class filter for ''section A'' to uncover patterns in the airfare. Similarly, use Stop filter with section B and for Section C, use the Season filter.
This project reflects my journey from Data Analysis to Machine Learning & from Deployment to Interactive Dashboard.
